P-\/\+CATS provides an alternative to simulating a large number of particles in Electric and Magnetic Fields.

P-\/\+CATS replaces traditional engines that utilise \mbox{\hyperlink{classParticle}{Particle}} in Cell (PIC) to optimise the computation of particle motions in Electric and Magnetic Fields.

P-\/\+CATS uses the \href{https://www.nature.com/articles/324446a0}{\texttt{ Barnes-\/\+Hut Algorithm}}, originally designed for N-\/body simulation of gravitational bodies, to separate particle clusters into separate 3D octets. Interactions between particles is then calculated by treating multipke particles far away enough as a single point-\/charge.\hypertarget{md_README_autotoc_md2}{}\doxysection{Installation}\label{md_README_autotoc_md2}
